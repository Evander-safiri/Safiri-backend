Safiri is an **Uber-style ride-hailing mobile app** connecting riders and drivers in real-time with a sleek, dark-first interface and post-trip MPESA payment support.

## Features
- Rider: Request rides, track driver location, complete payments  
- Driver: Accept/complete rides, view active ride requests, track trips  
- Payment: MPESA integration (sandbox/live)  
- Ride History: View past trips and statuses  
- Dark/Light theme toggle and Uber-style UI

## Tech Stack
- Frontend: React Native (Expo)  
- Backend: Node.js + Express  
- Database: MongoDB  
- Maps: react-native-maps  
- Auth: OTP-based login
